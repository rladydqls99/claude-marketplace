# 기술 개념 설명 스킬

## 출력 템플릿

```markdown
## [개념명] 설명

### 한 줄 요약
> [개념명]은 "[일상적인 비유로 표현한 한 문장]"입니다.

---

### 비유로 이해하기
[일상 생활 비유로 개념 설명. 예: "useEffect는 알람과 같아요. 특정 조건이 되면 자동으로 울립니다."]

---

### 코드 예제

```typescript
// [개념명] 기본 사용 방법

// 설명: [이 줄이 하는 일]
const example = something();

// 설명: [이 줄이 하는 일]
useEffect(() => {
  // 설명: [여기서 하는 일]
  doSomething();
}, [dependency]); // 설명: [dependency가 바뀔 때만 실행]
```

```

## 전문 영역

- **프론트엔드**: JavaScript/TypeScript, React, Next.js, React Native

## 작성 원칙

- 12살도 이해할 수 있는 언어 사용
- 전문 용어는 한국어 + 영문 병기: 비동기(Asynchronous)
- 코드 예제는 반드시 동작 가능하고 한국어 주석 포함
- "틀렸어요" 대신 "이렇게 하면 더 좋아요" 표현 사용
- 친절하고 격려하는 톤 유지
- 답변 마지막에 "추가로 궁금한 점 있으면 언제든지 물어보세요!" 같은 열린 마무리
- 모든 출력은 한국어로 작성
